Are you experiencing an annoying "Intelligent Work" pop-up that appears in the bottom right corner every time you open or switch between Microsoft 365 applications, such as Word, Excel, or Outlook? This notification is often triggered by the MSI Center software's AI Engine feature, particularly after a recent update. It promotes profile switching but can be disruptive during productivity tasks. This guide provides a detailed, step-by-step solution to disable the pop-up while keeping your system optimized, drawing from common resolutions for MSI hardware users running Microsoft 365 on Windows.

How to Stop the "Intelligent Work" Pop-Up in Microsoft 365 Apps

The "Intelligent Work" pop-up is typically linked to MSI's AI Engine, which automatically adjusts system profiles for tasks like office work. Disabling the animation notification in MSI Center resolves this without fully turning off the AI features, ensuring your MSI laptop or desktop continues to perform efficiently. Follow these steps to eliminate the interruption.

Step-by-Step Guide to Disable the Intelligent Work Notification

  1. Launch MSI Center: Search for "MSI Center" in the Windows Start menu using Windows key + S, or locate it in the system tray (bottom-right taskbar). Right-click the MSI Center icon and select "Open" if it's running in the background. If not installed, download the latest version from the official MSI website for your device model.
  2. Access the AI Engine Feature: Once MSI Center opens, navigate to the left sidebar and click on "AI Engine" (it may appear under "Features" or "System Tuner" depending on your version). Ensure the AI Engine is enabled, as we'll only modify the notification settings.
  3. Open AI Engine Settings: In the AI Engine dashboard, look for the status indicator showing "AI Engine: ON" (usually at the top or center). Click the gear icon (⚙️) next to it to open the detailed configuration options.
  4. Disable the Profile Switching Animation: In the settings window, scroll to the "Notifications" or "Visual Effects" section. Uncheck the box labeled "Show the profile switching animation when AI Engine initiates" or a similar option like "Display Intelligent Work pop-up." If you see a toggle for "Intelligent Work notifications," switch it to OFF.
  5. Apply Changes and Test: Click "Apply" or "Save" at the bottom of the window to confirm the changes. Close MSI Center, then restart your Microsoft 365 app (e.g., open Excel via Windows key + R, type excel, and press Enter). Switch between apps or minimize/restore windows to verify the pop-up no longer appears.
  6. Optional: Fully Disable AI Engine if Needed: If the pop-up persists or you prefer to turn off AI Engine entirely, return to the AI Engine dashboard and toggle the main switch to "OFF." This stops all automatic profile adjustments but may slightly impact battery life or performance tuning on MSI devices.

These steps are compatible with Windows 10 and 11, and MSI Center versions 2.0 or later. If your MSI Center interface differs slightly (e.g., due to a hardware-specific build), search within the app for "AI Engine" or "notifications" using the built-in search bar.

Troubleshooting: Why the "Intelligent Work" Pop-Up Still Appears After Disabling

If the pop-up continues despite following the steps above, it could stem from software conflicts, outdated drivers, or cached settings. Use this troubleshooting guide to identify and resolve underlying issues for a seamless Microsoft 365 experience on MSI systems.

Check for MSI Center Updates

Recent updates to MSI Center have been known to re-enable notifications. Open MSI Center, go to "Support" > "Live Update," and click "Scan for Updates." Install any available patches, then restart your computer (Windows key + X, select "Shut down or sign out" > "Restart"). Reapply the notification disable steps afterward.

Verify Microsoft 365 Integration and Conflicts

The pop-up may interact with Microsoft 365's own AI features like Copilot. In any Office app (e.g., Word), go to File > Options > General and ensure "Enable optional connected experiences" is unchecked if you suspect overlap. Additionally, run the Microsoft Support and Recovery Assistant: Download it from the official Microsoft site, select "Office" > "Performance," and follow the automated repair prompts.

Clear System Cache and Reset MSI Services

Cached animations can persist. Open Command Prompt as administrator (Windows key + X > "Windows Terminal (Admin)"), then run sfc /scannow to scan for corrupted files. For MSI-specific resets, in MSI Center, navigate to "Features" > "User Scenario" and reset profiles to default. If issues continue, uninstall and reinstall MSI Center via Settings > Apps > search for "MSI Center" > "Uninstall," then redownload from MSI's support page.

Advanced: Registry Edit for Persistent Notifications (Use with Caution)

For stubborn cases, edit the Windows Registry. Press Windows key + R, type regedit, and navigate to HKEY_LOCAL_MACHINE\SOFTWARE\MSI\AI Engine. Look for a DWORD value like "ShowAnimation" and set it to 0 (create it if missing, base 10). Restart your system. Warning: Back up the registry first via File > Export, as incorrect edits can cause system instability.

Hardware-Specific Considerations for MSI Laptops and Desktops

On MSI gaming laptops (e.g., GE, GS series), ensure Dragon Center (older version) isn't conflicting—uninstall it if present. For desktops, check BIOS settings: Restart and enter BIOS (Delete key during boot), go to "Advanced" > "Integrated Peripherals," and disable any AI-related enhancements if applicable. Update your BIOS from MSI's website for your motherboard model to prevent update-related bugs.

These troubleshooting tips address 90% of reported cases for "Intelligent Work" pop-ups in Microsoft 365 environments. If your setup involves custom MSI configurations, consult your device's manual for tailored AI Engine options.
